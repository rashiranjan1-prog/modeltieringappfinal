Upload your Excel workbook — both the matrix format and the standard format are supported. Tiering will be auto-computed after upload.

Supported Sheet Formats

Format A — Matrix (your current file)

Models are columns, parameters are rows. Scores are cell values (1–3).

Tiering_Method sheet

Parameter | Sub-Parameter | Description | Value Range | Weight | [Model 1] | [Model 2] | …

Each model column contains the score (1=Low, 2=Medium, 3=High) for that parameter.

Parameters sheet

Parameter | Sub-Parameter | Description | Low (1) | Medium (2) | High (3)

Models_List sheet

Sl. No. | Model Name | Risk Type

Model_tier sheet (optional — pre-assigned tiers)

Sl. No. | Model Name | Risk Type | Tier

Format B — Standard

One row per record in each sheet.

Models sheet

name | risk_type

Parameters sheet

group | sub_parameter | criteria | description | weight

Tiers sheet

name | lower_bound | upper_bound

Settings sheet

key | value

Keys: materiality_weight, criticality_weight, complexity_weight

Scores sheet (optional)

model_name | sub_parameter | level

level: 1=Low, 2=Medium, 3=High
